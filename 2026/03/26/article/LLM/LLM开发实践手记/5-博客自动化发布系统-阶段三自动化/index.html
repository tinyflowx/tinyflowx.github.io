1.1 方案对比

GitHub Actions 免费、配置简单、与博客同一仓库 仅支持 GitHub

六、常见问题

6.1 RSS 源失效

问题：某个 RSS 源突然返回 404

解决：

  1. 检查 RSS URL 是否变化
  2. 更新 config.js 中的 URL
  3. 如果无法恢复，从订阅源列表中移除

6.2 GitHub Actions 权限不足

问题：Permission denied when pushing

解决：确保 GIT_TOKEN 有仓库写入权限，建议使用 Fine-grained PAT 并授予写入权限。
